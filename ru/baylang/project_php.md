# Настройка PHP проекта

Создайте проект через composer
```
composer
```

Зайдите в проект:
```
cd project
```

Соберите докер контейнер:
```
docker build
```

Запустите докер контейнер:
```
docker run
```

Зайдите в докер контейнер:
```
docker exec
```

Скомпилируйте проект:
```
cd /var/www/html
baylang-php make_all
```
