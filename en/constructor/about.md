# BayLang Constructor. Website and CRM Builder

**BayLang Constructor** is a tool for creating software through a visual interface with the ability to edit BayLang source code.

**Key Features:**
- Develop cross-platform web applications and websites using drag-and-drop
- The constructor works directly with the program code
- FullStack technology for Frontend and Backend development
- Supports WordPress, PHP, VueJS
- Code export for use in other projects
